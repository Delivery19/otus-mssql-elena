Страховая компания.
Договор страхования заключается между страховой компанией и клиентом сроком не более одного года (но может быть и меньше). При заключении договора указывается вид страхования, страховая сумма, дата начала действия договора. 
Один клиент может заключить несколько договоров.
Один сотрудник работает с несколькими клиентами.
Страхование осуществляется на основе существующего вида услуг.

База данных приведена к 3 нормальной форме (находится во 2 нормальной форме и каждый неключевой атрибут не зависит от другого неключевого атрибута)
Основные таблицы - Operator, Client, Contract.
Поле PaspInfoID таблицы PaspInfo имеет связь один к одному с полем ClientID таблицы Client
Поле ClientID таблицы Contract является внешним ключом Поля ClientID таблицы Client
Поле typeID таблицы Contract является внешним ключом Поля typeID таблицы ContractType
Поле OperatorID таблицы Contract является внешним ключом Поля OperatorID таблицы Operator

=======================
В каждой таблице ограничения по последовательности

Ограничения
Таблица PaspInfo
Поле Unavailable, по умолчанию 0, т.е. паспорт действующий

Таблица Contract
Индекс по Клиентам в таблице Договоров.
Поле enddate - не более одного года с начальной датой по полю opendate
